### Run Project locally
1- download or clone the repo\
2- download python (3.8 or higher)\
3- install pipenv
```
pip install pipenv
```
4- navigate to project folder\
5- activate virtual environment
```
pipenv shell
```
6- install requirements
```
pipenv install
```
7- run the app
```
python app.py
```
